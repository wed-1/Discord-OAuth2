# Discord-OAuth2
Solving Discord login issues, using a OAuth2

**Dependencies:**
- `http`
- `fs`
- `url`
- `node-fetch`

**How to install?**
1. Clone the repo
2. Make Sure you have **nodejs & npm** installed
3. Open repo in VSCode Or any other editor
4. Using your terminal type: `npm i` to install dependecies

You're Ready to go..
